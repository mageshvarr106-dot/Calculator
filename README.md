# 🧮 Calculator App

A simple and interactive calculator built using HTML, CSS, and JavaScript.

## 🚀 Features
- Perform basic arithmetic operations (+, -, *, /)
- Real-time display updates
- Persistent data using LocalStorage
- Clean and minimal UI

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript

## 📸 Preview
A lightweight calculator designed to strengthen core JavaScript concepts like DOM manipulation and state handling.

## 📚 What I Learned
- Handling user input dynamically
- Updating UI using JavaScript
- Using localStorage to persist data
- Writing cleaner and structured code
