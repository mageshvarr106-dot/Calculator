# 🧮 Calculator App

A simple and interactive calculator built using HTML, CSS, and JavaScript.

## 🚀 Features
- Perform basic arithmetic operations (+, -, *, /)
- Real-time display updates
- Persistent data using LocalStorage
- Clean and minimal UI

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
